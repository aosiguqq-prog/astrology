# 画面式样书 · astrology（SPEC-U）

> SDD 阶段 4 产出物之一。本册规定画面布局、组件、交互、状态（空/错/载入态）与视觉规范。
> 编号前缀 `SPEC-U-xx`。视觉为「暗色神秘系」（discovery § 5 / § 6）。术语遵循 `discovery.md § 7`。
> 相关分册：行为见 `functional.md`（SPEC-F），数据见 `data.md`（SPEC-D），非功能（响应式断点、性能）见 `nfr.md`（SPEC-N）。

## SPEC-U-01 全局布局与暗色主题
- **来源**：FR-21；NFR-02；discovery § 5「全局导航」、§ 6-5「神秘暗色系响应式」
- **描述**：应用为单页应用，统一暗色主题，布局自上而下为：顶部全局导航（SPEC-U-02）→ 当前板块内容区 → 底部区域（含全局免责声明入口/版权，可选）。
  - **色板（固定基准，实现须落到具体色值）**：
    | 语义 | 用途 | 色值基准 |
    |------|------|----------|
    | 背景主色 | 页面/内容区底 | 深紫到深蓝渐变，基准 `#1A1033`（深紫）→ `#0D1B3E`（深蓝） |
    | 背景次色 | 卡片/面板底 | `#241640`（半透深紫） |
    | 主文字 | 正文 | `#EDE9F5`（近白） |
    | 次文字 | 说明/标签 | `#A99FC7`（淡紫灰） |
    | 装饰强调 | 强调、选中、星盘刻度、上升点 | 金色 `#E8C36B` |
    | 星光点缀 | 背景星点/分隔 | `#F5E7C4`（浅金） |
    | 危险/错误 | 错误提示文字 | `#F0868B`（柔红） |
  - **对比度**：主文字与背景主色对比度 ≥ 4.5:1（WCAG AA 正文，见 SPEC-N-02）。
- **验收/可测点**：
  1. 三板块页面均采用上述背景与文字色板，无浅色/白底页面。
  2. 金色仅用于装饰/强调/选中态，不用于大面积背景。
  3. 主文字对比度 ≥ 4.5:1。
- **边界与异常**：响应式断点见 SPEC-N-02；窄视口下布局纵向堆叠，无横向滚动溢出。

## SPEC-U-02 全局导航组件
- **来源**：FR-21；navigation.feature 全部四场景；SPEC-F-15
- **描述**：全局导航固定含且仅含三项「星盘 / 运势 / 百科」，桌面视口横向排列于顶部，移动视口可为顶部横排或底部标签栏（均须三项常驻可点）。
  - 当前板块项高亮：以金色（`#E8C36B`）标识文字/下划线/背景之一，其余两项为次文字色。
  - 首次加载默认高亮并呈现「星盘」。
  - 点击某项即切换到对应板块内容，当前板块标识随之更新，切换不触发整页刷新。
- **验收/可测点**：
  1. 导航恰含「星盘」「运势」「百科」三项，无第四项、无账号/登录项。
  2. 首次加载「星盘」高亮，星盘内容可见。
  3. 点击任一项，目标板块内容显示且该项变为高亮态，其余取消高亮。
  4. 切换后再切回，先前板块状态保持（SPEC-F-15：不重置已生成星盘/已选星座/已浏览词条）。
- **边界与异常**：三板块两两可达；导航项在所有视口尺寸下均可点击、无溢出遮挡。

## SPEC-U-03 出生信息表单
- **来源**：FR-01、FR-05；SPEC-F-01；实体 BirthInfo（SPEC-D-01）
- **描述**：星盘板块的输入区，含三个字段组件与「时间未知」开关及提交按钮：
  1. **出生日期**：日期选择器，年范围 1900 至当前年；必填标记。
  2. **出生时间**：时:分选择器（24h）；当「时间未知」开关开启时置灰禁用。
  3. **时间未知开关**：切换控件，标签「时间未知」；开启时禁用时间输入并在下方提示「将生成不含上升与宫位的降级星盘」。
  4. **出生地点**：文本输入 + 候选下拉；输入后触发解析（SPEC-A-01），同名多候选时显示候选列表供选择（SPEC-U-04）。
  5. **提交按钮**：文案「生成本命盘」。
  - **校验状态**：字段校验失败时，字段下方以错误色（`#F0868B`）显示对应提示文案（文案见 SPEC-F-01）：日期空→「出生日期为必填项」；日期越界→「出生日期超出可用范围」；地点空→「请选择出生地点」；地点无法识别→「出生地点无法识别，请重新选择城市」。
- **验收/可测点**：
  1. 表单恰含日期、时间、地点三字段 + 时间未知开关 + 提交按钮，无其它出生字段。
  2. 未填日期提交，日期字段下方显示「出生日期为必填项」，不进入生成。
  3. 开关开启后时间输入不可编辑，且可成功提交进入降级盘（SPEC-U-05 降级态）。
  4. 地点无法识别时显示「出生地点无法识别，请重新选择城市」，不生成星盘。
- **边界与异常**：错误提示与对应字段就近显示；提交按钮在校验失败时不触发星历计算。

## SPEC-U-04 城市候选选择
- **来源**：FR-02；SPEC-F-02；SPEC-A-01（candidates）
- **描述**：当地点解析返回多个候选（同名城市）时，在地点输入下方展示候选列表，每项显示「城市名 · 省/州」（用 `province` 消歧），用户须点选唯一候选后方可提交生成。
- **验收/可测点**：
  1. 同名城市输入后，展示 ≥2 个候选项，各含可区分的省/州信息。
  2. 未选定候选前点击「生成本命盘」，提示「请选择出生地点」，不生成。
  3. 选定候选后，该候选的坐标/时区用于计算（SPEC-A-02）。
- **边界与异常**：精确唯一命中时不展示候选列表，直接可提交。

## SPEC-U-05 星盘视图
- **来源**：FR-08、FR-03；SPEC-F-07/F-04；实体 NatalChart（SPEC-D-02）
- **描述**：星盘板块在生成成功后呈现圆形本命盘图 + 行星落座清单：
  1. **圆形星盘图**：
     - 外圈：黄道十二星座刻度，按 C-SIGN 等分 12 段，各段标星座符号，金色刻度线。
     - 十大行星：按黄经定位，标 C-PLANET 行星符号。
     - 十二宫：宫头分隔线划分 12 扇区，标宫序号 1~12。
     - 上升点（ASC）：盘左侧水平位置以金色标记，标注上升星座。
     - 相位连线：盘内圈连线表示相位，按 AspectType 区分线型/颜色（如合相实线金、对分虚线红、三分实线蓝、四分虚线橙、六分点线紫——线型须彼此可区分且提供图例）。
  2. **行星落座清单**：列 10 行，每行「行星名 | 所在星座 | 所在宫位」。
  3. **降级态**：`hasTime=false` 时——不绘制上升点标记与宫位分隔线；星座刻度与行星符号照常绘制；清单「宫位」列每行显示「不可用」；盘上/清单附近显示说明「出生时间未知，上升与宫位不可用」。
- **验收/可测点**：
  1. 完整盘：可见 12 星座刻度、10 行星符号、12 宫分隔、1 个上升点标记、相位连线（≥0 条，附图例）、10 行清单（三列非空）。
  2. 降级盘：无上升标记、无宫位分隔；清单宫位列全为「不可用」；显示降级说明文案。
  3. 无相位时不绘制连线，其余元素照常。
- **边界与异常**：星盘图在窄视口下等比缩放，保持可读（SPEC-N-01/N-02）。

## SPEC-U-06 星盘要素解读卡
- **来源**：FR-09、FR-19、FR-22；SPEC-F-09/F-13；实体 EncyclopediaEntry（SPEC-D-05）
- **描述**：在星盘视图上，用户点击某行星落座（图上符号或清单行）可就地弹出解读卡：
  1. **触发方式**：点击行星符号或清单某行。
  2. **内容结构**：卡片标题（如「太阳 落 天蝎」）+ 该落座星座的百科解释（`explanation`，与 SPEC-F-11 同源）+ 关键词。
  3. **免责声明位置**：卡片底部固定显示「仅供娱乐参考」，与解读内容同屏可见（SPEC-F-13）。
  4. 关闭卡片返回星盘，不离开访客模式、不跳转整页。
- **验收/可测点**：
  1. 点击行星落座弹出解读卡，展示的解释与对应星座百科词条一致。
  2. 卡片内可见「仅供娱乐参考」。
  3. 关闭卡片后星盘视图状态不变。
- **边界与异常**：降级盘不提供「宫位」解读入口（宫位不可用），行星落座星座解读仍可用。

## SPEC-U-07 运势视图
- **来源**：FR-10、FR-11、FR-13、FR-14、FR-22；SPEC-F-10/F-12/F-13；实体 HoroscopeContent（SPEC-D-04）
- **描述**：运势板块含星座选择、每日/每周切换、运势维度卡与降级空态：
  1. **星座选择**：12 星座选择控件（列表/网格），当前所选星座高亮。
  2. **周期切换**：「每日 / 每周」切换控件。
  3. **日期标签**：每日显示当前系统日期；每周显示本周周一~周日区间。
  4. **运势维度卡**：每日至少爱情/事业/健康三张维度卡，每卡含维度名 + 文案（有评分则显示 1~5）；每周显示整体概述卡。
  5. **免责声明**：运势内容区显示「仅供娱乐参考」，与内容同屏（SPEC-F-13）。
  6. **降级空态**：运势不可用（SPEC-F-12）时，不显示任何维度卡，改为空态卡显示「运势内容暂时无法获取，请稍后再试」+「重试」按钮。
- **验收/可测点**：
  1. 选择某星座每日运势，显示当前系统日期标签 + ≥3 张维度卡（含爱情/事业/健康）+「仅供娱乐参考」。
  2. 切每周运势，显示本周区间 + 概述 +「仅供娱乐参考」。
  3. 所选星座与展示运势星座一致；换星座后内容更新为该星座。
  4. API 不可用时显示降级空态文案与重试按钮，不显示任何运势内容与免责声明。
- **边界与异常**：点击重试重新请求（SPEC-A-03）；成功则退出空态显示内容。

## SPEC-U-08 百科视图
- **来源**：FR-15、FR-16、FR-17、FR-18；SPEC-F-11；实体 EncyclopediaEntry（SPEC-D-05）
- **描述**：百科板块含词条列表、搜索框与词条详情：
  1. **搜索框**：顶部输入框，占位「搜索行星或星座」；输入即时或提交后触发搜索（SPEC-A-04）。
  2. **词条列表**：默认展示 22 条（可分「星座 / 行星」两组或统一列表并标类别徽标），每项显示名称 + 类别徽标 + 符号。
  3. **词条详情**：点击某词条进入详情，结构为「名称 + 符号 + 象征 + 关键词 + 解释」四类内容分区展示。
  4. **搜索空结果**：无匹配时列表区显示「未找到相关词条」空态。
- **验收/可测点**：
  1. 默认列表含 22 条，每项标注类别（星座/行星）。
  2. 搜索「天蝎」结果含天蝎词条且标为「星座」；搜索「太阳」结果含太阳词条标为「行星」。
  3. 词条详情展示符号、象征、关键词、解释四区，均非空。
  4. 搜索无匹配词，显示「未找到相关词条」。
- **边界与异常**：空搜索框显示全部 22 条（不显示空结果提示，SPEC-F-11）。

## SPEC-U-09 空态 / 错误态 / 加载态
- **来源**：FR-04、FR-05、FR-13、FR-18；NFR-07；SPEC-F-06/F-12
- **描述**：统一三态视觉规范：
  1. **加载态**：星历计算/运势请求进行中，显示暗色系加载指示（如金色旋转星点），不显示半成品内容。
  2. **错误/拒绝态**：以错误色文案就近显示，文案严格取自式样（表单校验见 SPEC-U-03；运势降级见 SPEC-U-07；搜索空结果见 SPEC-U-08）。
  3. **空态**：无内容时显示占位说明（如百科搜索前提示、运势未选星座提示），不留空白裸区。
- **验收/可测点**：
  1. 运势请求期间显示加载指示，不展示旧内容。
  2. 各错误文案与 SPEC-F 定义逐字一致（「出生地点无法识别，请重新选择城市」「出生日期为必填项」「运势内容暂时无法获取，请稍后再试」「未找到相关词条」）。
  3. 三态均采用暗色系视觉，无白底裸露。
- **边界与异常**：加载态须有超时归宿（运势 8 秒后进入降级空态，SPEC-N-05）。
