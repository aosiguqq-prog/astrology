# 🌙 占星 App

一个纯前端、离线可用的占星网页应用，支持本命星盘生成、每日/每周运势与行星星座百科。

**在线访问**：[https://aosiguqq-prog.github.io/astrology/](https://aosiguqq-prog.github.io/astrology/)

---

## 功能

- **本命星盘**：输入出生日期、时间与出生地，生成可视化星盘（十大行星、十二宫、上升点、相位连线）
- **每日 / 每周运势**：12 星座各一套运势，含爱情、事业、健康三维度与星级评分，纯离线可用
- **行星 & 星座百科**：22 条预置词条，支持关键词搜索与星盘联动跳转
- **访客模式**：无需注册登录，刷新即清空，不存储任何个人数据

## 技术栈

| 层 | 技术 |
|---|---|
| 框架 | React 18 + TypeScript |
| 构建 | Vite 6 |
| 样式 | Tailwind CSS v4 |
| 状态 | Zustand（纯内存，不持久化）|
| 星历计算 | 离线轨道近似算法（纯 JS）|
| 地理编码 | 离线城市库（115 城，含山东全省）|
| 测试 | Vitest，136 条，行覆盖率 97.68% |

## 本地运行

```bash
npm install
npm run dev        # 开发服务器 → http://localhost:5173/astrology/
npm run build      # 生产构建
npm run test       # 运行测试
```

## 部署

```bash
npm run deploy     # 构建并推送到 GitHub Pages
```

## 已知限制

- 行星位置为近似计算，误差 ≤ 2°，仅供占星参考
- 宫位系统为等宫近似（非严格 Placidus）
- 城市库覆盖全球 115 座主要城市，不含乡镇
