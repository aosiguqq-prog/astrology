# Changelog

本项目所有值得记录的变更都会写入本文件。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.1.0] - 2026-07-13

### Changed
- 运势数据来源由外部第三方 API（aztro）改为**本地预设内容**：12 星座 × 每日/每周各一套，含爱情/事业/健康三维度 + 星级评分。纯离线、永远可用，不再出现「运势内容暂时无法获取」的降级空态（修复用户反馈的运势不可用问题，同时消除 review M1/M2 的生产代理缺失与外部供应链风险）。
- `horoscope/index.ts` 默认 provider 切换为 `createPresetProvider`；`aztro-provider.ts` 保留导出，作为日后换回真实 API 的备选。

### Added
- 山东省城市数据大幅补全：新增 14 个地级市（淄博、枣庄、东营、烟台、潍坊、济宁、泰安、威海、日照、临沂、德州、聊城、滨州、菏泽）+ 26 个主要县级市/市辖区（胶州、即墨、平度、莱西、龙口、招远、莱州、蓬莱、海阳、荣成、乳山、文登、青州、诸城、寿光、高密、昌邑、邹城、曲阜、滕州、新泰、肥城、乐陵、禹城、临清、章丘）。山东省城市数由 2 个增至 42 个，城市库总量由 75 增至 115。
- 新增预设运势与山东城市的回归测试（+5 条，共 136 条）。

## [1.0.0] - 2026-07-12

### Added
- 本命盘生成：输入出生日期/时间/地点，生成完整可视化星盘（十大行星、十二宫、上升点、相位）
- 运势板块：接入第三方运势 API，展示每日/每周运势，含降级处理
- 行星/星座百科：22 条预置词条，支持浏览与关键词搜索
- 星盘-百科联动：从星盘要素跳转到对应百科词条
- 神秘暗色主题 UI（深紫/深蓝背景 + 金色装饰）
- 响应式布局，支持桌面与移动浏览器
- 访客模式，无需注册

### Technical
- React 18 + TypeScript + Vite 静态构建
- 纯离线星历计算（无外部 API 依赖）
- Zustand 会话内存状态管理（无持久化）
- 131 条单元测试，行覆盖率 97.68%
