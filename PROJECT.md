# astrology

> 网页版占星应用

## 一句话需求
面向个人用户的网页占星应用，支持出生星盘生成（本命盘）、每日/每周运势推送、以及星座与行星百科，采用神秘暗色风格。

## 背景 / 动机
- 给谁用：对占星感兴趣的普通个人用户
- 核心价值：输入出生信息即可获得完整星盘图，并结合每日运势与知识库加深对自身的了解

## 需求摘要（来自 intake 阶段）
| 维度 | 内容 |
|------|------|
| 平台 | 网页版（浏览器访问） |
| 核心功能 | 星盘生成（本命盘）、每日/每周运势、行星/星座百科 |
| UI 风格 | 神秘暗色系（深紫/深蓝背景，金色/星光装饰） |
